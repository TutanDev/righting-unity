com.tutan.functional

Functional

Null-safe, railway-oriented types for Unity. Optional<T>, Result<T>, and composable extensions that make errors impossible to ignore.

Example.cs 
// Optional<T> — no more null checks
Optional<Player> player = FindPlayer(id);

player
    .Then(p => p.Inventory)
    .Then(inv => inv.GetWeapon("sword"))
    .Match(
        onNone: () => Log("No sword found"),
        onSome: sword => Equip(sword)
    );

// Result<T> — errors you can't ignore
Result<Order> result = PlaceOrder(request);

result
    .Then(order => Validate(order))
    .Then(order => Save(order))
    .Match(
        onError: err => Reporter.Error(err.Message),
        onSuccess: order => Confirm(order)
    );

Optional<T>

Null-safe value wrapper. IsSome/IsNone, pattern matching, implicit conversions from T and None.

Result<T>

Success-or-error railway. Chain operations with .Then(), recover with .Or() and .OrElse().

Error Type

Immutable, composable errors with inner error chains. Implicit conversion from string.

Async Support

Full async/await support for both Optional and Result. Chain async operations seamlessly.

Utilities

Five domain-agnostic services that every Unity project needs. Drop in the installer, configure via ScriptableObjects, and ship.

MessageBus

Zero-allocation pub/sub. Struct messages passed by ref. Immediate and queued dispatch. Thread-safe enqueue for worker threads.

CommandBus EventBus Zero GC

Reporter

Leveled logging with tag filtering and scoped timing. Compile-time stripping via [Conditional]. Built-in metric accumulation.

Timer

Handle-based callback scheduler. One-shot and repeating. Scaled or unscaled time. Pool-based with generation counters for safe cancellation.

Pooled Handle-Safe Repeating

AudioPlayer

Pooled AudioSource playback with fluent SoundParams. Spatial audio, steal-oldest overflow, handle-based lifecycle.
